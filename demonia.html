Voor het vak Frontend Development, moesten wij twee pagina's van een website naar keuze namaken met HTML, CSS en Javascript. Verdere criteria waren dat de pagina's toegangelijk moesten zijn, er minimaal één micro-interactie in zit, en deze responsive is voor kleinere schermen.

Voor mijn project heb ik de website van schoenenmerk Demonia gekozen. Hiervan heb ik de Homepagina en de "Shop all" pagina nagemaakt. Bekijk de website
